## 🎁 SIMPLE

<br/>
• SIMPLE은 만족스러운 캠핑을 위한 아이템을 판매하는 e-commerce입니다.<br/>
• SIMPLE은 캠핑에 감성을 더해주고 그 공간을 더욱 가치있게 바꿔줍니다.<br/>
• SIMPLE은 가심비를 추구하는 MZ세대를 위한 선물을 제공합니다. 가심비는 비용과 상관없이 만족스러운 것을 구매하는 소비 행태입니다. <br/>
